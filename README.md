# プラたべる（PlaTaberu）

## 概要
「プラたべる」は、**海洋プラスチック問題を楽しく学びながら解決に貢献できるアプリ**です。  
ユーザが砂浜の砂を回収し、プラスチックを分離・検出することで、キャラクターが成長します。

環境問題への関心を高めると同時に、実際の回収活動をゲーム体験として設計しています。

---

## 開発背景
周防大島の観光資源であるビーチでは、漂着ごみによる景観悪化や、海洋生物への影響が問題となっていました。

そこで、
- 「環境問題を身近に感じてもらう」
- 「楽しみながら行動につなげる」

ことを目的に、本アプリを開発しました。

---

## システム概要

本システムは以下の流れで動作します：

1. 砂浜の砂（プラスチックを含む）を回収
2. 水と混ぜて専用装置で撹拌
3. 比重差によりプラスチックを水面に浮上させる
4. カメラで撮影し、AI（YOLOv7）で検出
5. 回収量・種類に応じてキャラクターが成長

---

## 特徴

### ゲーム性
  - 回収したプラスチックに応じてキャラクターが成長
  - 継続的なモチベーションを提供

### AIによる物体検出
  - YOLOv7を用いてプラスチックを検出
  - 種類・量を定量的に評価

### 社会課題へのアプローチ
  - 海洋汚染問題を体験的に学べる設計
